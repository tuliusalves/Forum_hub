# Forum_hub
<p>O projeto Fórum Hub Api consistiu na construção de uma API utilizando Java, Spring, MySQL. O objetivo principal foi Desenvolver uma API para um fórum de discussão chamado Fórum Hub, utilizando as melhores práticas de desenvolvimento de software, incluindo um CRUD completo, paginação, autenticação via JWT e validações de dados.</p>
<h2>Pré-requisitos</h2>
<ul>
  <li>Java 17 ou superior instalado na máquina</li>
  <li>MySQL instalado na máquina</li>
</ul>
<h2>Modo de usar</h2>
<uL>
<li>Instale o MySQL na máquina</li>
<li>Crie uma um banco de dados no MySQL com o nome forumhub</li>
<li>Configure as variáveis de ambiente no seu sistema operacional</li>
<li>DB_HOST com o valor localhost</li>
<li>DB_NAME com o valor forumhub</li>
<li>DB_PASSWORD com o valor de sua senha do banco do MySQL</li>
<li>DB_USER com o valor do seu usuário do banco do MySQL</li>
<li>JWT_SECRET com uma senha qualquer para a criptografia o JWT</li>
<li>Instale o IntelliJ IDEA e rode o projeto</li>
</uL>
<h2>Tecnologias usadas</h2>
<ul>
<li>Java - Linguagem de programação</li>
<li>Spring Boot - Framework</li>
<li>Maven - Gerenciador de dependências</li>
<li>IntelliJ IDEA - IDE</li>
<li>MySQL - Banco de dados</li>
<li>JWT (JSON Web Token) Biblioteca para a geração do Token</li>
<li>Swagger Biblioteca para documentação da api</li>
</ul>
